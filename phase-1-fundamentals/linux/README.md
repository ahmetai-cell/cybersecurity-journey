# Linux Temelleri

> Siber güvenlikte Linux şart. Kali Linux, Ubuntu Server öğren.
> Süre: 2-3 hafta (networking ile paralel)

---

## Konu Listesi

- [ ] Dosya sistemi yapısı
- [ ] Temel komutlar (ls, cd, cp, mv, rm, cat, grep)
- [ ] Dosya izinleri (chmod, chown)
- [ ] Ağ komutları (ip, netstat, ss, ping, traceroute, nmap)
- [ ] Servis yönetimi (systemctl)
- [ ] Log analizi (/var/log/)
- [ ] Bash scripting temelleri
- [ ] Kullanıcı yönetimi
